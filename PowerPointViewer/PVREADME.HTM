Acerca do PowerPoint Viewer

O Microsoft Office PowerPoint Viewer permite-lhe visualizar apresentações diversificadas criadas no Microsoft Office PowerPoint 2003. O PowerPoint Viewer suporta a visualização de apresentações criadas no PowerPoint 97, PowerPoint 2000 e PowerPoint 2002. O PowerPoint Viewer também suporta que se abram apresentações do PowerPoint protegidas por palavra-passe.

Requisitos do sistema

  • PC com um processador a 200 MHz ou superior
  • Microsoft Internet Explorer 5 ou posterior
  • Sistemas operativos suportados: Microsoft Windows 98 Segunda Edição, Windows Millennium Edition, Windows 2000 com o Service Pack 3, Windows XP ou Windows Server 2003.
  • Memória RAM:
    • Para o Windows 98 SE, Windows ME: 32 MB de RAM
    • Para o Windows 2000 com o Service Pack 3: 64 MB de RAM
    • Para o Windows XP, Windows Server 2003: 128 MB de RAM
  • 5 MB de espaço disponível no disco rígido
  • Monitor Super VGA (800 x 600) ou com resolução superior

Problemas com vários idiomas

O PowerPoint Viewer não suporta o Microsoft Office Proofing Tools.

O PowerPoint Viewer não suporta apresentações em idiomas bidirecionais criadas no PowerPoint 97.

Trabalhar com o PowerPoint Viewer

O PowerPoint Viewer não apresenta animações de diagrama correctamente. Em vez de apresentar as animações, o diagrama é apresentado na sua totalidade.

O PowerPoint Viewer não suporta que se abram apresentações com a gestão de direitos da informação (IRM, Information Rights Management).

O PowerPoint Viewer não suporta que se executem macros e programas ou que se abram objectos ligados ou incorporados.

O PowerPoint Viewer não suporta que se apresentem os seguintes formatos gráficos quando ligados numa apresentação: .eps, .pct, .emz, .cdr, .cgm, .wpg e .wmz.
